🍀 GOODLUCK $ — Architecture Overview

A web app that uses an AI Agent to analyze Nasdaq 100 stocks and recommend short-term swing trades.
Owner: hooky2019 · Live URL: https://goodluck.streamlit.app

One-line explanation:
You tap Refresh → AI reads the market → screens 100 stocks → picks the top 3 trade ideas → returns a full plan (entry, stop, targets, position size).

2. What Happens on a Refresh

When you tap 🔄 Refresh picks, the AI calls tools in this order:

sequenceDiagram participant U as 👤 User participant A as 🤖 AI Agent participant D as 📊 Data Tools participant Y as 🌐 Yahoo / News U->>A: Tap Refresh picks A->>D: get_market_context() D->>Y: Fetch SPY / QQQ / VIX Y-->>D: Index data D-->>A: Regime (risk-on/off/chop) A->>D: screen_universe() D->>Y: Scan 100 Nasdaq stocks Y-->>D: Per-ticker scores D-->>A: Top 15 shortlist loop For top 5 candidates A->>D: get_technicals(ticker) A->>D: get_catalysts(ticker) D-->>A: EMA/RSI/MACD/ATR + earnings date end loop For top 3 finalists A->>D: get_news(ticker) A->>D: get_options_snapshot(ticker) A->>D: get_reddit_mentions(ticker) D-->>A: News / options / social sentiment end A->>A: Synthesize + emit JSON report A-->>U: Render top 3 pick cards
Key point: The AI is not following a hardcoded sequence — it decides which tool to call next based on what it has learned so far. That dynamic decision-making is what makes this an AI Agent rather than a script.

8. Secrets Management

API keys must never be committed to GitHub. Three layers of protection:

  1. Local dev: Keys live in .streamlit/secrets.toml — listed in .gitignore, so git never touches them.
  2. Cloud runtime: Keys are pasted into Streamlit Cloud → Settings → Secrets — stored encrypted.
  3. Code access: Python reads with st.secrets.get("GROQ_API_KEY") — works the same way in both environments.

The two required keys:

  • GROQ_API_KEY — for Llama 3.3 70B (free at console.groq.com)
  • ALPHA_VANTAGE_KEY — for news sentiment (free at alphavantage.co)

      "setup": "breakout / pullback / mean-reversion / momentum",
      "thesis": "why this, why now (2-3 sentences)",
      "entry_zone": [128.5, 129.8],          ← Entry range
      "stop": 125.1,                          ← Stop loss
      "stop_basis": "1.5x ATR / structural support",

      "rr_ratio": 2.8,                        ← Risk/reward (min 2:1)
      "position_size_pct": 3.5,               ← % of account
      "position_size_dollars": 3500,
      "risks": ["what would invalidate the trade"],
      "options_play": {                       ← Optional defined-risk play
        "type": "long call",
        "strike": 130,

      }
    }
  ]
