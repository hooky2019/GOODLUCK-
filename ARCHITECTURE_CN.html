2. 一次刷新发生了什么 (Refresh Flow)

当你按下 🔄 Refresh picks 按钮时，AI 会按顺序调用 10 个工具:

sequenceDiagram participant U as 👤 用户 participant A as 🤖 AI Agent participant D as 📊 Data Tools participant Y as 🌐 Yahoo / News U->>A: 按下 Refresh picks A->>D: get_market_context() D->>Y: 抓取 SPY / QQQ / VIX Y-->>D: 大盘走势数据 D-->>A: 市场情绪 (risk-on/off/chop) A->>D: screen_universe() D->>Y: 扫描 100 只 Nasdaq 股票 Y-->>D: 每只股票评分 D-->>A: Top 15 候选名单 loop 对 Top 5 候选 A->>D: get_technicals(ticker) A->>D: get_catalysts(ticker) D-->>A: EMA/RSI/MACD/ATR + 财报日 end loop 对最强 3 只 A->>D: get_news(ticker) A->>D: get_options_snapshot(ticker) A->>D: get_reddit_mentions(ticker) D-->>A: 新闻 / 期权 / 社交情绪 end A->>A: 综合分析 + 出 JSON 报告 A-->>U: 显示 Top 3 picks 卡片
关键点: AI 不是按固定流程走，而是自己决定什么时候调用哪个工具 (Tool Calling)。这就是 AI Agent 跟传统脚本的最大区别。

8. 安全和密钥管理 (Secrets Management)

密钥 (API Keys) 绝对不能 写进代码里推到 GitHub。本项目用三层保护:

  1. 本地开发: 密钥放在 .streamlit/secrets.toml — 这个文件在 .gitignore 里，不会被 git 跟踪。
  2. 云端运行: 密钥粘到 Streamlit Cloud → Settings → Secrets — 加密存储。
  3. 代码访问: Python 用 st.secrets.get("GROQ_API_KEY") 读取 — 本地读文件，云端读云端存储。

需要的两个密钥:

  • GROQ_API_KEY — 给 Llama 3.3 70B 用 (免费，从 console.groq.com 拿)
  • ALPHA_VANTAGE_KEY — 新闻情绪用 (免费，从 alphavantage.co 拿)
